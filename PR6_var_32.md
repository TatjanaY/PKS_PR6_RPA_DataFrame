## **Часть 1**

Компания **ServiceRequests** занимается обработкой заявок клиентов на техническое обслуживание и ремонт.   
Ежедневно поступают десятки заявок: часть новых, часть — в работе, а часть — уже завершена.  
Руководство хочет контролировать заявки со статусом "новая", отслеживать приоритеты и выявлять заявки, требующие срочного внимания.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о заявках через **DataFrame** в **Puzzle RPA**, отфильтровать заявки по статусу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

**Шаги:**

1. **Создать исходный файл `requests.xlsx` с 10 записями:**

```python
# requests.xlsx

| R003       | "Сидоров С."  | "Москва"          | "низкий" | "новая"        | 2026-02-25   |
| R004       | "Кузнецов К." | "Казань"          | "высокий"| "новая"        | 2026-02-27   |
| R005       | "Морозова М." | "Москва"          | "средний"| "завершена"    | 2026-02-20   |
| R006       | "Новиков Н."  | "Санкт-Петербург" | "низкий" | "новая"        | 2026-02-28   |

| R008       | "Смирнов С."  | "Казань"          | "средний"| "завершена"    | 2026-02-26   |
| R009       | "Егоров Е."   | "Москва"          | "высокий"| "новая"        | 2026-03-01   |
| R010       | "Лебедева Л." | "Санкт-Петербург" | "низкий" | "завершена"    | 2026-02-25   |
```

* **Типы данных:**

```python
request_id: str
client_name: str
city: str
priority: str  # "высокий", "средний", "низкий"
request_status: str  # "новая", "в работе", "завершена"
request_date: date
```

2. **Считать исходный файл `requests.xlsx` в объект в DataFrame `requests_df`**


3. **Добавить столбец `is_urgent` — флаг срочности (высокий приоритет и статус "новая"):**

```python
requests_df["is_urgent"] = (requests_df["priority"] == "высокий") & (requests_df["request_status"] == "новая")
```

4. **Отфильтровать заявки со статусом "новая":**

```python
new_requests_df = FilterDataFrame(
    table=requests_df,
    filter_condition='request_status == "новая"'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python

NotifyUser(message="Новые заявки успешно сохранены в new_requests.xlsx!")
```

## **Часть 2**

После фильтрации новых заявок руководство хочет проанализировать время реакции специалистов.   
Существует второй файл `technicians.xlsx`, в котором указаны данные о назначенных техниках и времени их реакции.   
Нужно объединить данные и выделить заявки, на которые время реакции превышает 24 часа, а результат сохранить и показать пользователю.

---

1. **Создать файл `technicians.xlsx` с 8 записями:**

```python
# technicians.xlsx
| request_id | technician_name | response_time | assignment_date |
|------------|-----------------|---------------|-----------------|
| R001       | "Техник А."     | 12            | 2026-02-28      |
| R002       | "Техник Б."     | 36            | 2026-03-01      |
| R003       | "Техник В."     | 18            | 2026-02-25      |
| R004       | "Техник А."     | 48            | 2026-02-27      |
| R005       | "Техник Г."     | 8             | 2026-02-20      |
| R006       | "Техник Б."     | 30            | 2026-02-28      |
| R007       | "Техник В."     | 15            | 2026-03-02      |
| R009       | "Техник А."     | 28            | 2026-03-01      |
```

* **Типы данных:**

```python
request_id: str
technician_name: str
response_time: int  # время реакции в часах
assignment_date: date
```

2. **Считать исходный файл `technicians.xlsx` в объект в DataFrame `technicians_df`**

3. **Объединить `new_requests_df` и `technicians_df` по `request_id` (inner join):**

```python
combined_df = JoinDataFrames(
    table1=new_requests_df,
    table2=technicians_df,
    keys1=["request_id"],
    keys2=["request_id"],
    join_type="inner"
)
```

4. **Выбрать заявки с временем реакции > 24 часов:**

```python
slow_response_df = FilterDataFrame(
    table=combined_df,
    filter_condition='response_time > 24'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="slow_response_requests.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по заявкам с медленной реакцией сохранена в slow_response_requests.xlsx!")
```
