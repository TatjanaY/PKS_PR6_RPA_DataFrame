## **Часть 2**

После фильтрации по категории руководство хочет проверить полноту оплат.   
Существует второй файл `payments.xlsx`, в котором указаны фактические поступления средств.   
Нужно объединить данные и выделить заказы, где сумма оплаты меньше суммы заказа (недооплата), а результат сохранить и показать пользователю.

1. **Создать файл `payments.xlsx` с 8 записями:**


* **Типы данных:**

```python
order_id: str
payment_date: date
payment_amount: int
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `smartphone_orders_df` и `payments_df` по `order_id`:**

```python
combined_df = JoinDataFrames(
    table1=smartphone_orders_df,
    table2=payments_df,
    keys1=["order_id"],
    keys2=["order_id"],
    join_type="inner"
)
```

4. **Добавить расчет `remaining_amount`:**

```python
combined_df["remaining_amount"] = combined_df["amount"] - combined_df["payment_amount"]
```

5. **Выбрать заказы с неполной оплатой:**

```python
underpaid_df = FilterDataFrame(
    table=combined_df,
    filter_condition='payment_amount < amount'
)
```
