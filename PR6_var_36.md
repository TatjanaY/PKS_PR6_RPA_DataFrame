## **Часть 1**

Компания **EventManager** занимается организацией мероприятий и конференций по городам России.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о регистрациях через **DataFrame** в **Puzzle RPA**, отфильтровать регистрации по городу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации регистраций в Москве руководство хочет проанализировать статусы оплат.   
Существует второй файл `payments.xlsx`, в котором указаны данные о платежах участников.   
Нужно объединить данные и выделить регистрации со статусом оплаты "не оплачено", а результат сохранить и показать пользователю.

---

1. **Создать файл `payments.xlsx` с 8 записями:**

```python
# payments.xlsx
| reg_id | payment_date | payment_amount | payment_method | payment_status |
|--------|--------------|----------------|----------------|----------------|
| R001   | 2026-03-01   | 5000           | "карта"        | "оплачено"     |
| R002   | 2026-03-02   | 0              | null           | "не оплачено"  |
| R003   | 2026-02-28   | 0              | null           | "не оплачено"  |
| R004   | 2026-03-03   | 7000           | "банковский"   | "оплачено"     |
| R006   | 2026-03-04   | 0              | null           | "не оплачено"  |
| R007   | 2026-03-05   | 5000           | "карта"        | "оплачено"     |
| R009   | 2026-03-06   | 0              | null           | "не оплачено"  |
| R010   | 2026-03-01   | 5000           | "карта"        | "оплачено"     |
```

* **Типы данных:**

```python
reg_id: str
payment_date: date
payment_amount: int  # сумма платежа (0 если нет оплаты)
payment_method: str  # "карта", "банковский", null
payment_status: str  # "оплачено", "не оплачено"
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `moscow_regs_df` и `payments_df` по `reg_id` (inner join):**

```python
combined_df = JoinDataFrames(
    table1=moscow_regs_df,
    table2=payments_df,
    keys1=["reg_id"],
    keys2=["reg_id"],
    join_type="inner"
)
```

4. **Выбрать регистрации со статусом оплаты "не оплачено":**

```python
unpaid_regs_df = FilterDataFrame(
    table=combined_df,
    filter_condition='payment_status == "не оплачено"'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="unpaid_moscow_registrations.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по неоплаченным регистрациям в Москве сохранена в unpaid_moscow_registrations.xlsx!")
```
