## **Часть 1**

Компания **ExpenseMonitor** занимается учётом расходов сотрудников в крупных организациях.   
В системе регистрируются данные о служебных расходах: командировки, офисные нужды, обучение и другие категории.  
Руководство хочет контролировать расходы на командировки, отслеживать суммы трат и выявлять операции, требующие дополнительного согласования.

Ваша задача как RPA-разработчика — автоматизировать работу с данными расходов через **DataFrame** в **Puzzle RPA**, отфильтровать расходы по типу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

```python
expenses_df["is_recent"] = (today - expenses_df["expense_date"]).days <= 7
```

## **Часть 2**

После фильтрации расходов на командировки руководство хочет проанализировать данные сотрудников, совершивших эти траты.   
Существует второй файл `employees.xlsx`, в котором указаны данные о сотрудниках и их должностях.   
Нужно объединить данные и выделить расходы на сумму более 5000 рублей, а результат сохранить и показать пользователю.

* **Типы данных:**

```python
emp_id: str
emp_name: str
department: str
position: str
manager: str
approval_limit: int   # лимит на самостоятельное согласование
```

2. **Считать исходный файл `employees.xlsx` в объект в DataFrame `employees_df`**

3. **Объединить `business_trip_df` и `employees_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=business_trip_df,
    table2=employees_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Добавить расчет `exceeds_limit`:**

```python
combined_df["exceeds_limit"] = combined_df["expense_amount"] > combined_df["approval_limit"]
```

5. **Выбрать расходы на сумму > 5000:**

```python
high_expenses_df = FilterDataFrame(
    table=combined_df,
    filter_condition='expense_amount > 5000'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python

SaveExcel(file_path="high_business_expenses.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по крупным расходам на командировки сохранена в high_business_expenses.xlsx!")
```
