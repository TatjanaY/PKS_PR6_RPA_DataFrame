## **Часть 1**

Компания **SupplyTrack** занимается управлением цепочками поставок и логистикой для розничных сетей по городам России.   
Ежедневно обрабатываются данные о доставках товаров от различных поставщиков: часть доставок выполнена в срок, часть — с задержкой, а часть — отменена.  
Руководство хочет контролировать поставки со статусом "задержка", анализировать причины просрочек и формировать отчёт по проблемным поставщикам.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о поставках через **DataFrame** в **Puzzle RPA**, отфильтровать задержанные доставки, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации задержанных поставок руководство хочет проанализировать данные по поставщикам.   
Существует второй файл `suppliers.xlsx`, в котором указаны рейтинги поставщиков и количество дней задержки.   
Нужно объединить данные и выделить поставщиков с задержкой более 2 дней, а результат сохранить и показать пользователю.

---

1. **Создать файл `suppliers.xlsx` с 8 записями:**

```python
# suppliers.xlsx
| delivery_id | supplier_rating | delay_days |
|-------------|-----------------|------------|
| D001        | 5               | 0          |
| D002        | 3               | 4          |
| D003        | 5               | 0          |
| D004        | 2               | 5          |
| D006        | 4               | 0          |
| D007        | 3               | 3          |
| D009        | 4               | 1          |
| D010        | 5               | 0          |
```

* **Типы данных:**

```python
delivery_id: str
supplier_rating: int  # рейтинг от 1 до 5
delay_days: int       # количество дней задержки
```

2. **Считать исходный файл `suppliers.xlsx` в объект в DataFrame `suppliers_df`**

3. **Объединить `delayed_deliveries_df` и `suppliers_df` по `delivery_id`:**

```python
combined_df = JoinDataFrames(
    table1=delayed_deliveries_df,
    table2=suppliers_df,
    keys1=["delivery_id"],
    keys2=["delivery_id"],
    join_type="inner"
)
```

4. **Выбрать поставки с задержкой более 2 дней:**

```python
critical_delays_df = FilterDataFrame(
    table=combined_df,
    filter_condition='delay_days > 2'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="critical_delays.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по критическим задержкам поставок сохранена в critical_delays.xlsx!")
```
