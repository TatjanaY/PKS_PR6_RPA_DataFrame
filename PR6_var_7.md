# Вариант 7 - WarehousePro: контроль остатков и задержек поставок

---

## **Часть 1**

Компания **WarehousePro** управляет складскими остатками товаров для розничной сети по городам России.   
На складе хранятся электроника, одежда, товары для дома и другие категории.  
Руководство хочет контролировать товары с низким остатком, отслеживать поставки от поставщиков и выявлять позиции, требующие срочного пополнения.

Ваша задача как RPA-разработчика — автоматизировать работу с остатками через **DataFrame** в **Puzzle RPA**, отфильтровать товары по количеству, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

**Шаги:**

# inventory.xlsx
| item_id | item_name        | category    | quantity | warehouse_city | last_update |

| I005    | "Чайник электрический"| "дом"   | 15       | "Москва"       | 2026-03-07  |
| I006    | "Наушники Wireless"| "электроника"| 30    | "Санкт-Петербург"| 2026-03-11|
| I007    | "Куртка осенняя" | "одежда"     | 5        | "Москва"       | 2026-03-06  |

| I009    | "Планшет Mini"   | "электроника"| 22       | "Москва"       | 2026-03-10  |

* **Типы данных:**

```python
item_id: str
item_name: str
category: str      # "электроника", "одежда", "обувь", "дом"
quantity: int
warehouse_city: str
last_update: date
```

2. **Считать исходный файл `inventory.xlsx` в объект в DataFrame `inventory_df`**


3. **Добавить столбец `is_low_stock` — флаг низкого остатка:**

```python
inventory_df["is_low_stock"] = inventory_df["quantity"] < 20
```

4. **Отфильтровать товары с количеством < 20:**

```python
low_stock_df = FilterDataFrame(
    table=inventory_df,
    filter_condition='quantity < 20'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python

NotifyUser(message="Товары с низким остатком успешно сохранены в low_stock.xlsx!")
```

## **Часть 2**

После фильтрации товаров с низким остатком руководство хочет проверить статус поставок от поставщиков.   
Существует второй файл `suppliers.xlsx`, в котором указаны данные о поставках и задержках.   
Нужно объединить данные и выделить товары, где есть задержка поставки, а результат сохранить и показать пользователю.

---

1. **Создать файл `suppliers.xlsx` с 8 записями:**

```python
# suppliers.xlsx
| item_id | supplier_name | delivery_date | expected_date | delivery_delay |
|---------|---------------|---------------|---------------|----------------|
| I002    | "TechSupply"  | 2026-03-12    | 2026-03-10    | 2              |
| I003    | "FashionLog"  | 2026-03-15    | 2026-03-12    | 3              |
| I005    | "HomeGoods"   | 2026-03-11    | 2026-03-11    | 0              |
| I007    | "FashionLog"  | 2026-03-14    | 2026-03-10    | 4              |
| I008    | "HomeGoods"   | 2026-03-13    | 2026-03-12    | 1              |
| I010    | "SportWear"   | 2026-03-16    | 2026-03-13    | 3              |
| I001    | "TechSupply"  | 2026-03-10    | 2026-03-10    | 0              |
| I004    | "SportWear"   | 2026-03-11    | 2026-03-09    | 2              |
```

* **Типы данных:**

```python
item_id: str
supplier_name: str
delivery_date: date
expected_date: date
delivery_delay: int   # дней задержки
```

2. **Считать исходный файл `suppliers.xlsx` в объект в DataFrame `suppliers_df`**

3. **Объединить `low_stock_df` и `suppliers_df` по `item_id`:**

```python
combined_df = JoinDataFrames(
    table1=low_stock_df,
    table2=suppliers_df,
    keys1=["item_id"],
    keys2=["item_id"],
    join_type="inner"
)
```

4. **Добавить расчет `urgency_score`:**

```python
combined_df["urgency_score"] = combined_df["quantity"] * combined_df["delivery_delay"]
```

5. **Выбрать товары с задержкой поставки > 0:**

```python
delayed_supplies_df = FilterDataFrame(
    table=combined_df,
    filter_condition='delivery_delay > 0'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="delayed_supplies.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по товарам с задержкой поставки сохранена в delayed_supplies.xlsx!")
```
