## **Часть 1**

Компания **TransportSchedule** занимается планированием и контролем транспортных рейсов по городам России.   
В системе регистрируются данные о маршрутах, статусах рейсов, времени отправления и прибытия.  
Руководство хочет контролировать рейсы с задержками, отслеживать причины сбоев и выявлять маршруты, требующие корректировки расписания.

Ваша задача как RPA-разработчика — автоматизировать работу с данными рейсов через **DataFrame** в **Puzzle RPA**, отфильтровать маршруты по статусу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

```python
routes_df["is_significant_delay"] = routes_df["delay_minutes"] > 30
```

2. **Считать исходный файл `drivers.xlsx` в объект в DataFrame `drivers_df`**

3. **Объединить `delayed_routes_df` и `drivers_df` по `route_id`:**

```python
combined_df = JoinDataFrames(
    table1=delayed_routes_df,
    table2=drivers_df,
    keys1=["route_id"],
    keys2=["route_id"],
    join_type="inner"
)
```

4. **Добавить расчет `delay_impact_score`:**

```python
combined_df["delay_impact_score"] = combined_df["delay_minutes"] / combined_df["rating"]
```

5. **Выбрать рейсы с задержкой > 30 минут:**

```python
critical_delays_df = FilterDataFrame(
    table=combined_df,
    filter_condition='delay_minutes > 30'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python

SaveExcel(file_path="critical_delays.xlsx")
```
