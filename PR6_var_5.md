Существует второй файл `payments.xlsx`, в котором указаны фактические поступления средств.   
Нужно объединить данные и выделить заказы, где сумма оплаты меньше суммы заказа (частичная оплата), а результат сохранить и показать пользователю.

1. **Создать файл `payments.xlsx` с 8 записями:**

```python

| O006     | 2026-02-28   | 12000          |

| O009     | 2026-03-01   | 3000           |
| O010     | 2026-02-25   | 7200           |
| O001     | 2026-02-28   | 3500           |
```

* **Типы данных:**

```python
order_id: str
payment_date: date
payment_amount: int
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `high_value_orders_df` и `payments_df` по `order_id`:**

```python
combined_df = JoinDataFrames(
    table1=high_value_orders_df,
    table2=payments_df,
    keys1=["order_id"],
    keys2=["order_id"],

)
```

4. **Добавить расчет `remaining_amount`:**

```python

```

5. **Выбрать заказы с частичной оплатой:**

```python
partial_payment_df = FilterDataFrame(
    table=combined_df,
    filter_condition='payment_amount < amount'

```
