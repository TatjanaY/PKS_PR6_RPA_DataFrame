## **Часть 1**

Компания **OnlineSchool** занимается организацией онлайн-занятий для студентов по различным направлениям.   
Ежедневно обрабатываются данные о студентах: часть проходит курсы по веб-разработке, часть — по дизайну, а часть — по маркетингу.  
Руководство хочет контролировать студентов, изучающих курс "Web", отслеживать их посещаемость и выявлять учащихся с большим количеством пропусков.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о студентах через **DataFrame** в **Puzzle RPA**, отфильтровать студентов по курсу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации студентов курса "Web" руководство хочет проанализировать данные о посещаемости занятий.   
Существует второй файл `attendance.xlsx`, в котором указано количество пропущенных уроков каждым студентом.   
Нужно объединить данные и выделить студентов, пропустивших более 3 занятий, а результат сохранить и показать пользователю.

---

1. **Создать файл `attendance.xlsx` с 8 записями:**

```python
# attendance.xlsx
| student_id | total_lessons | attended_lessons | missed_lessons | last_attendance |
|------------|---------------|------------------|----------------|-----------------|
| S001       | 20            | 18               | 2              | 2026-03-10      |
| S002       | 18            | 15               | 3              | 2026-03-09      |
| S003       | 20            | 14               | 6              | 2026-03-08      |
| S004       | 15            | 12               | 3              | 2026-03-11      |
| S005       | 20            | 19               | 1              | 2026-03-10      |
| S007       | 18            | 13               | 5              | 2026-03-09      |
| S009       | 20            | 11               | 9              | 2026-03-07      |
| S010       | 22            | 20               | 2              | 2026-03-11      |
```

* **Типы данных:**

```python
student_id: str
total_lessons: int      # общее количество уроков
attended_lessons: int   # посещённые уроки
missed_lessons: int     # пропущенные уроки
last_attendance: date   # дата последнего посещения
```

2. **Считать исходный файл `attendance.xlsx` в объект в DataFrame `attendance_df`**

3. **Объединить `web_students_df` и `attendance_df` по `student_id`:**

```python
combined_df = JoinDataFrames(
    table1=web_students_df,
    table2=attendance_df,
    keys1=["student_id"],
    keys2=["student_id"],
    join_type="inner"
)
```

4. **Выбрать студентов с количеством пропущенных уроков > 3:**

```python
absent_students_df = FilterDataFrame(
    table=combined_df,
    filter_condition='missed_lessons > 3'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="absent_students.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по студентам с большим количеством пропусков сохранена в absent_students.xlsx!")
```
