# Вариант 12 - StorageManager: анализ товаров с низким остатком

---

## **Часть 1**

Компания **StorageManager** управляет складскими запасами товаров для розничной торговли по городам России.   
Ежедневно обрабатываются данные об остатках товаров на складах: часть товаров имеет достаточный запас, часть — низкий остаток, а часть — требует срочного пополнения.  
Руководство хочет контролировать товары с количеством менее 15 единиц, отслеживать поставщиков и выявлять позиции с низким рейтингом поставщиков.

Ваша задача как RPA-разработчика — автоматизировать работу с данными об остатках через **DataFrame** в **Puzzle RPA**, отфильтровать товары с низким остатком, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

**Шаги:**

1. **Создать исходный файл `inventory.xlsx` с 10 записями:**

```python
# inventory.xlsx
| product_id | product_name      | warehouse       | quantity | category      | last_update |
|------------|-------------------|-----------------|----------|---------------|-------------|

| P002       | "Смартфон Xiaomi" | "Склад-СПб"     | 8        | "электроника" | 2026-03-09  |
| P003       | "Наушники Sony"   | "Склад-МСК"     | 12       | "аксессуары"  | 2026-03-08  |
| P004       | "Клавиатура Logitech" | "Склад-КЗН" | 30       | "аксессуары"  | 2026-03-10  |
| P005       | "Мышь Razer"      | "Склад-МСК"     | 5        | "аксессуары"  | 2026-03-07  |
| P006       | "Монитор Dell"    | "Склад-СПб"     | 14       | "электроника" | 2026-03-09  |

| P008       | "ОЗУ 16GB"        | "Склад-КЗН"     | 10       | "комплектующие"| 2026-03-08 |

```

* **Типы данных:**

```python
product_id: str
product_name: str
warehouse: str
quantity: int
category: str  # "электроника", "аксессуары", "комплектующие"
last_update: date
```

2. **Считать исходный файл `inventory.xlsx` в объект в DataFrame `inventory_df`**


3. **Добавить столбец `is_low_stock` — флаг низкого остатка:**

```python
inventory_df["is_low_stock"] = inventory_df["quantity"] < 15
```

4. **Отфильтровать товары с количеством менее 15:**

```python
low_stock_df = FilterDataFrame(
    table=inventory_df,
    filter_condition='quantity < 15'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python

NotifyUser(message="Товары с низким остатком успешно сохранены в low_stock.xlsx!")
```

## **Часть 2**

После фильтрации товаров с низким остатком руководство хочет проанализировать данные по поставщикам.   
Существует второй файл `suppliers.xlsx`, в котором указаны рейтинги поставщиков и сроки поставки.   
Нужно объединить данные и выделить товары, поставляемые поставщиками с рейтингом ниже 3, а результат сохранить и показать пользователю.

---

1. **Создать файл `suppliers.xlsx` с 8 записями:**

```python
# suppliers.xlsx
| product_id | supplier_name   | supplier_rating | delivery_days |
|------------|-----------------|-----------------|---------------|
| P001       | "ООО ТехноПоставка" | 5             | 2             |
| P002       | "ИП ГаджетТрейд"    | 2             | 5             |
| P003       | "ООО ТехноПоставка" | 4             | 3             |
| P004       | "ЗАО АксессуарПлюс" | 3             | 4             |
| P005       | "ИП ГаджетТрейд"    | 2             | 6             |
| P006       | "ООО ТехноПоставка" | 5             | 2             |
| P008       | "ЗАО КомплектСервис"| 1             | 7             |
| P009       | "ИП ГаджетТрейд"    | 2             | 5             |
```

* **Типы данных:**

```python
product_id: str
supplier_name: str
supplier_rating: int  # рейтинг от 1 до 5
delivery_days: int    # срок поставки в днях
```

2. **Считать исходный файл `suppliers.xlsx` в объект в DataFrame `suppliers_df`**

3. **Объединить `low_stock_df` и `suppliers_df` по `product_id` (left join):**

```python
combined_df = JoinDataFrames(
    table1=low_stock_df,
    table2=suppliers_df,
    keys1=["product_id"],
    keys2=["product_id"],
    join_type="left"
)
```

4. **Выбрать товары с рейтингом поставщика < 3:**

```python
critical_suppliers_df = FilterDataFrame(
    table=combined_df,
    filter_condition='supplier_rating < 3'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="critical_suppliers.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по товарам с низко рейтинговыми поставщиками сохранена в critical_suppliers.xlsx!")
```
