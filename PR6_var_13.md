## **Часть 2**

После фильтрации сотрудников IT-отдела руководство хочет проанализировать фактическое рабочее время.   
Существует второй файл `work_hours.xlsx`, в котором указаны данные о отработанных часах за неделю.   
Нужно объединить данные и выделить сотрудников с переработками (более 45 часов в неделю), а результат сохранить и показать пользователю.

---

1. **Создать файл `work_hours.xlsx` с 8 записями:**

```python
# work_hours.xlsx
| emp_id | week_start  | hours_week | overtime_hours | project_code |
|--------|-------------|------------|----------------|--------------|
| E001   | 2026-03-03  | 48         | 8              | "PROJ-101"   |
| E003   | 2026-03-03  | 52         | 12             | "PROJ-205"   |
| E005   | 2026-03-03  | 45         | 5              | "PROJ-101"   |
| E007   | 2026-03-03  | 40         | 0              | "PROJ-310"   |
| E009   | 2026-03-03  | 46         | 6              | "PROJ-205"   |
| E002   | 2026-03-03  | 38         | 0              | "SUP-045"    |
| E004   | 2026-03-03  | 42         | 2              | "MKT-088"    |
| E006   | 2026-03-03  | 44         | 4              | "SAL-122"    |
```

* **Типы данных:**

```python
emp_id: str
week_start: date
hours_week: int
overtime_hours: int
project_code: str
```

2. **Считать исходный файл `work_hours.xlsx` в объект в DataFrame `work_hours_df`**

3. **Объединить `it_employees_df` и `work_hours_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=it_employees_df,
    table2=work_hours_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Добавить расчет `overtime_rate`:**

```python
combined_df["overtime_rate"] = combined_df["overtime_hours"] / combined_df["hours_week"] * 100
```

5. **Выбрать сотрудников с переработками (часов в неделю > 45):**

```python
overtime_employees_df = FilterDataFrame(
    table=combined_df,
    filter_condition='hours_week > 45'

```
