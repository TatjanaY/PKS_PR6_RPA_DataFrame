# Вариант 11 - LogisticsStock: контроль сроков хранения товаров на складе

---

## **Часть 1**

Компания **LogisticsStock** управляет складскими запасами товаров с ограниченным сроком годности по городам России.   
На складе хранятся продукты питания, косметика, бытовая химия и другие товары со сроком хранения.  
Руководство хочет контролировать товары с истекающим сроком годности, отслеживать остатки и выявлять позиции, требующие срочной реализации.

Ваша задача как RPA-разработчика — автоматизировать работу с остатками через **DataFrame** в **Puzzle RPA**, отфильтровать товары по сроку годности, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

* **Типы данных:**

```python
item_id: str
item_name: str
category: str      # "продукты", "косметика", "химия"
quantity: int
warehouse_city: str
expiry_date: date
```

2. **Считать исходный файл `inventory.xlsx` в объект в DataFrame `inventory_df`**


3. **Добавить столбец `expiry_days` — дней до истечения срока:**

```python
inventory_df["expiry_days"] = (inventory_df["expiry_date"] - today).days
```

4. **Отфильтровать товары со сроком годности < 7 дней:**

```python
expiring_soon_df = FilterDataFrame(

```

## **Часть 2**

После фильтрации товаров с истекающим сроком руководство хочет проверить наличие остатков на складе.   
Существует второй файл `expiry.xlsx`, в котором указаны данные о сроках хранения и статусах товаров.   
Нужно объединить данные и выделить товары, которые ещё есть в наличии и требуют срочной реализации, а результат сохранить и показать пользователю.

---

1. **Создать файл `expiry.xlsx` с 8 записями:**

```python
# expiry.xlsx
| item_id | expiry_date | days_remaining | storage_condition | quality_status |
|---------|-------------|----------------|-------------------|----------------|
| I001    | 2026-03-15  | 5              | "норма"           | "годен"        |
| I002    | 2026-03-12  | 2              | "норма"           | "годен"        |
| I004    | 2026-03-10  | 0              | "требуется проверка"| "под вопросом"|
| I006    | 2026-03-11  | 1              | "норма"           | "годен"        |
| I009    | 2026-03-13  | 3              | "норма"           | "годен"        |
| I003    | 2026-06-20  | 102            | "норма"           | "годен"        |
| I005    | 2026-08-15  | 158            | "норма"           | "годен"        |
| I007    | 2026-05-30  | 81             | "норма"           | "годен"        |
```

* **Типы данных:**

```python
item_id: str
expiry_date: date
days_remaining: int
storage_condition: str   # "норма", "требуется проверка"
quality_status: str      # "годен", "под вопросом"
```

2. **Считать исходный файл `expiry.xlsx` в объект в DataFrame `expiry_df`**

3. **Объединить `expiring_soon_df` и `expiry_df` по `item_id`:**

```python
combined_df = JoinDataFrames(
    table1=expiring_soon_df,
    table2=expiry_df,
    keys1=["item_id"],
    keys2=["item_id"],
    join_type="inner"
)
```

4. **Добавить расчет `urgency_level`:**

```python
combined_df["urgency_level"] = combined_df["quantity"] * (7 - combined_df["days_remaining"])
```

5. **Выбрать товары с остатком > 30:**

```python
available_expiring_df = FilterDataFrame(
    table=combined_df,
    filter_condition='quantity > 30'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="available_expiring.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по товарам с истекающим сроком и наличием сохранена в available_expiring.xlsx!")
```
