## **Часть 2**

После фильтрации по городу руководство хочет проанализировать возвраты товаров.   
Существует второй файл `returns.xlsx`, в котором указаны заявки на возврат от клиентов.   
Нужно объединить данные (учитывая, что не по всем заказам есть возвраты) и выделить заказы, где возврат ожидается, а результат сохранить и показать пользователю.

---

1. **Создать файл `returns.xlsx` с 8 записями:**

```python
# returns.xlsx
| order_id | return_date | return_status | refund_amount |
|----------|-------------|---------------|---------------|
| O002     | 2026-03-05  | "ожидается"   | 22000         |
| O004     | 2026-03-03  | "одобрен"     | 12000         |
| O006     | 2026-03-06  | "ожидается"   | 18000         |
| O008     | 2026-03-04  | "отклонен"    | 0             |
| O010     | 2026-03-05  | "ожидается"   | 14000         |
| O001     | 2026-03-02  | "одобрен"     | 15000         |
| O003     | 2026-03-01  | "ожидается"   | 8000          |
| O005     | 2026-02-28  | "отклонен"    | 0             |
```

* **Типы данных:**

```python
order_id: str
return_date: date
return_status: str   # "ожидается", "одобрен", "отклонен"
refund_amount: int
```

2. **Считать исходный файл `returns.xlsx` в объект в DataFrame `returns_df`**

3. **Объединить `spb_orders_df` и `returns_df` по `order_id`:**

```python
combined_df = JoinDataFrames(
    table1=spb_orders_df,
    table2=returns_df,
    keys1=["order_id"],
    keys2=["order_id"],
    join_type="left"
)
```

4. **Добавить расчет `days_since_return`:**

```python
combined_df["days_since_return"] = (today - combined_df["return_date"]).days
```

5. **Выбрать заказы со статусом возврата "ожидается":**

```python
pending_returns_df = FilterDataFrame(
    table=combined_df,
    filter_condition='return_status == "ожидается"'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="pending_returns.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по ожидаемым возвратам сохранена в pending_returns.xlsx!")
```
