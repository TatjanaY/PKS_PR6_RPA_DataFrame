## **Часть 2**

После фильтрации заказов в обработке руководство хочет проверить наличие задолженностей.   
Существует второй файл `payments.xlsx`, в котором указаны фактические поступления средств.   
Нужно объединить данные и выделить заказы, где сумма оплаты меньше суммы заказа (недооплата), а результат сохранить и показать пользователю.

---

1. **Создать файл `payments.xlsx` с 7 записями:**

```python
# payments.xlsx
| order_id | payment_date | payment_amount |
|----------|--------------|----------------|
| O001     | 2026-02-28   | 3500           |
| O002     | 2026-03-01   | 3000           |
| O003     | 2026-02-25   | 2800           |
| O004     | 2026-02-27   | 5000           |
| O006     | 2026-02-28   | 3300           |
| O007     | 2026-03-02   | 4000           |
| O009     | 2026-03-01   | 2500           |
```

* **Типы данных:**

```python
order_id: str
payment_date: date
payment_amount: int
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `processing_orders_df` и `payments_df` по `order_id`:**

```python
combined_df = JoinDataFrames(
    table1=processing_orders_df,
    table2=payments_df,
    keys1=["order_id"],
    keys2=["order_id"],
    join_type="inner"
)
```

4. **Добавить расчет `remaining_amount`:**

```python
combined_df["remaining_amount"] = combined_df["amount"] - combined_df["payment_amount"]
```

5. **Выбрать заказы с задолженностью >= 1:**

```python
unpaid_df = FilterDataFrame(
    table=combined_df,
    filter_condition='remaining_amount >= 1'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="unpaid_orders.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по заказам с задолженностью сохранена в unpaid_orders.xlsx!")
```
