## **Часть 2**

После фильтрации по Казани руководство хочет проверить статусы отправок заказов.   
Существует второй файл `shipments.xlsx`, в котором указаны данные о доставке.   
Нужно объединить данные и выделить заказы со статусом "задержка", а результат сохранить и показать пользователю.

---

1. **Создать файл `shipments.xlsx` с 8 записями:**

```python
# shipments.xlsx
| order_id | shipment_date | shipment_status |
|----------|---------------|-----------------|
| O001     | 2026-02-28    | "доставлено"    |
| O002     | 2026-03-01    | "в пути"        |
| O003     | 2026-02-25    | "задержка"      |
| O004     | 2026-02-27    | "доставлено"    |
| O006     | 2026-02-28    | "задержка"      |
| O007     | 2026-03-02    | "в пути"        |
| O008     | 2026-02-26    | "доставлено"    |
| O010     | 2026-02-25    | "задержка"      |
```

* **Типы данных:**

```python
order_id: str
shipment_date: date
shipment_status: str  # "доставлено", "в пути", "задержка"
```

2. **Считать исходный файл `shipments.xlsx` в объект в DataFrame `shipments_df`**

3. **Объединить `kazan_orders_df` и `shipments_df` по `order_id` (left join):**

```python
combined_df = JoinDataFrames(
    table1=kazan_orders_df,
    table2=shipments_df,
    keys1=["order_id"],
    keys2=["order_id"],
    join_type="left"
)
```

4. **Выбрать заказы со статусом отправки "задержка":**

```python
delayed_df = FilterDataFrame(
    table=combined_df,
    filter_condition='shipment_status == "задержка"'

```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python

NotifyUser(message="Аналитика по заказам с задержкой доставки сохранена в delayed_shipments.xlsx!")
```
