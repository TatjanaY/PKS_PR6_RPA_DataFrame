## **Часть 1**

Компания **StaffPerformance** занимается мониторингом эффективности работы сотрудников в различных отделах.   
Ежедневно обрабатываются данные о сотрудниках: часть показывает высокие результаты, часть — средние, а часть — требует дополнительного обучения.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о сотрудниках через **DataFrame** в **Puzzle RPA**, отфильтровать сотрудников по отделу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации сотрудников отдела продаж руководство хочет проанализировать их эффективность.   
Существует второй файл `performance.xlsx`, в котором указаны показатели эффективности каждого сотрудника.   
Нужно объединить данные и выделить сотрудников с показателем эффективности ниже 60, а результат сохранить и показать пользователю.

---

1. **Создать файл `performance.xlsx` с 8 записями:**

```python
# performance.xlsx
| emp_id | performance_score | sales_count | last_review_date |
|--------|-------------------|-------------|------------------|
| E001   | 75                | 45          | 2026-03-01       |
| E002   | 88                | 62          | 2026-03-05       |
| E003   | 65                | 38          | 2026-02-28       |
| E004   | 52                | 28          | 2026-03-03       |
| E005   | 45                | 22          | 2026-02-25       |
| E007   | 68                | 41          | 2026-03-08       |
| E009   | 38                | 15          | 2026-03-02       |
| E010   | 72                | 35          | 2026-03-06       |
```

* **Типы данных:**

```python
emp_id: str
performance_score: int  # оценка эффективности от 0 до 100
sales_count: int        # количество продаж
last_review_date: date
```

2. **Считать исходный файл `performance.xlsx` в объект в DataFrame `performance_df`**

3. **Объединить `sales_employees_df` и `performance_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=sales_employees_df,
    table2=performance_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Выбрать сотрудников с показателем эффективности < 60:**

```python
low_performance_df = FilterDataFrame(
    table=combined_df,
    filter_condition='performance_score < 60'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="low_performance_sales.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по сотрудникам с низкой эффективностью сохранена в low_performance_sales.xlsx!")
```
