## **Часть 1**

Компания **WorkLog** занимается учётом задач и проектов сотрудников в крупных организациях.   
В системе регистрируются задачи разных статусов: новые, в работе, завершённые и просроченные.  
Руководство хочет контролировать задачи, находящиеся в активной работе, отслеживать сроки выполнения и выявлять задачи, требующие внимания.

Ваша задача как RPA-разработчика — автоматизировать работу с задачами через **DataFrame** в **Puzzle RPA**, отфильтровать задачи по статусу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.


2. **Считать исходный файл `tasks.xlsx` в объект в DataFrame `tasks_df`**


3. **Добавить столбец `days_elapsed` — дней с начала задачи:**

```python
tasks_df["days_elapsed"] = (today - tasks_df["start_date"]).days
```

## **Часть 2**

Существует второй файл `employees.xlsx`, в котором указаны данные о сотрудниках.   
Нужно объединить данные и выделить задачи, которые просрочены более чем на 2 дня, а результат сохранить и показать пользователю.

* **Типы данных:**

```python

rating: float
```

2. **Считать исходный файл `employees.xlsx` в объект в DataFrame `employees_df`**

3. **Объединить `active_tasks_df` и `employees_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=active_tasks_df,
    table2=employees_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Добавить расчет `overdue_days`:**

```python
combined_df["overdue_days"] = (today - combined_df["deadline"]).days
```

5. **Выбрать задачи с просрочкой > 2 дней:**

```python
overdue_tasks_df = FilterDataFrame(
    table=combined_df,
    filter_condition='overdue_days > 2'
)
```

```python

NotifyUser(message="Аналитика по просроченным задачам сохранена в overdue_tasks.xlsx!")
```
