## **Часть 1**

Компания **OfficeAttendance** занимается учётом посещаемости сотрудников в распределённых офисах по городам России.   
В системе регистрируются данные о присутствии, опозданиях и отсутствиях сотрудников.  
Руководство хочет контролировать посещаемость в московском офисе, отслеживать пропуски и выявлять сотрудников с частыми отсутствиями.

Ваша задача как RPA-разработчика — автоматизировать работу с данными посещаемости через **DataFrame** в **Puzzle RPA**, отфильтровать записи по офису, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

## **Часть 2**

После фильтрации записей по московскому офису руководство хочет проанализировать количество пропусков по сотрудникам.   
Существует второй файл `employees.xlsx`, в котором указаны данные о сотрудниках и их суммарных показателях.   
Нужно объединить данные и выделить сотрудников с более чем 3 днями отсутствий, а результат сохранить и показать пользователю.

---

1. **Создать файл `employees.xlsx` с 8 записями:**

```python
| E007   | "Фролова Ф."| "IT"      | "тестировщик"| 4                 | 3         | 1             |
```

* **Типы данных:**

```python
emp_id: str
emp_name: str
department: str
position: str
total_absent_days: int
sick_days: int
personal_days: int
```

2. **Считать исходный файл `employees.xlsx` в объект в DataFrame `employees_df`**

3. **Объединить `moscow_attendance_df` и `employees_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=moscow_attendance_df,
    table2=employees_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Добавить расчет `absence_rate`:**

```python
combined_df["absence_rate"] = combined_df["total_absent_days"] / 22 * 100  # % от рабочих дней в месяце
```

5. **Выбрать сотрудников с отсутствиями > 3 дней:**

```python
frequent_absent_df = FilterDataFrame(
    table=combined_df,
    filter_condition='total_absent_days > 3'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python

SaveExcel(file_path="frequent_absent.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по сотрудникам с частыми отсутствиями сохранена в frequent_absent.xlsx!")
```
