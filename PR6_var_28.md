## **Часть 1**

Компания **PaymentAudit** занимается аудитом финансовых операций и контролем выставленных счетов.   
Ежедневно обрабатываются данные о счетах: часть оплачена, часть — в процессе оплаты, а часть — ожидает платежа.  
Руководство хочет контролировать счета со статусом "выставлен", отслеживать сроки оплаты и выявлять потенциальные проблемы с платежами.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о счетах через **DataFrame** в **Puzzle RPA**, отфильтровать счета по статусу, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации выставленных счетов руководство хочет проанализировать данные о платежах.   
Существует второй файл `payments.xlsx`, в котором указаны фактические поступления средств.   
Нужно объединить данные (left join) и выделить счета, по которым не поступило платежей (payment_amount == 0), а результат сохранить и показать пользователю.

---

1. **Создать файл `payments.xlsx` с 8 записями:**

```python
# payments.xlsx
| invoice_id | payment_date | payment_amount | payment_method |
|------------|--------------|----------------|----------------|
| INV001     | 2026-03-10   | 150000         | "банковский"   |
| INV002     | 2026-03-05   | 0              | null           |
| INV003     | 2026-03-08   | 0              | null           |
| INV004     | 2026-03-09   | 200000         | "банковский"   |
| INV005     | 2026-03-01   | 0              | null           |
| INV006     | 2026-03-11   | 0              | null           |
| INV008     | 2026-03-07   | 65000          | "карта"        |
| INV010     | 2026-03-06   | 130000         | "банковский"   |
```

* **Типы данных:**

```python
invoice_id: str
payment_date: date
payment_amount: int  # сумма платежа (0 если нет оплаты)
payment_method: str  # "банковский", "карта", null
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `issued_invoices_df` и `payments_df` по `invoice_id` (left join):**

```python
combined_df = JoinDataFrames(
    table1=issued_invoices_df,
    table2=payments_df,
    keys1=["invoice_id"],
    keys2=["invoice_id"],
    join_type="left"
)
```

4. **Выбрать счета без платежей (payment_amount == 0):**

```python
unpaid_invoices_df = FilterDataFrame(
    table=combined_df,
    filter_condition='payment_amount == 0'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="unpaid_invoices.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по неоплаченным счетам сохранена в unpaid_invoices.xlsx!")
```
