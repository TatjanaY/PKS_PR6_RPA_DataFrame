## **Часть 1**

Компания **RetailWarehouse** управляет сетью складов и логистических центров по городам России.

Руководство хочет контролировать перемещения товаров в конкретные города, отслеживать статусы доставок и выявлять проблемные маршруты.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о перемещениях через **DataFrame** в **Puzzle RPA**, отфильтровать перемещения по городу назначения, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

## **Часть 2**

После фильтрации перемещений в Москву руководство хочет проанализировать статусы доставок.   
Существует второй файл `warehouses.xlsx`, в котором указаны данные о складах-получателях и статусах перемещений.   
Нужно объединить данные и выделить перемещения со статусом "задержка", а результат сохранить и показать пользователю.

---

1. **Создать файл `warehouses.xlsx` с 7 записями:**

```python
# warehouses.xlsx
| move_id | warehouse_name | move_status | capacity |
|---------|----------------|-------------|----------|
| M001    | "Склад-МСК-Центр" | "доставлено" | 500      |
| M003    | "Склад-МСК-Центр" | "задержка"   | 500      |
| M005    | "Склад-МСК-Север" | "отменено"   | 300      |
| M006    | "Склад-МСК-Центр" | "задержка"   | 500      |
| M007    | "Склад-МСК-Юг"    | "в пути"     | 400      |
| M009    | "Склад-МСК-Центр" | "задержка"   | 500      |
| M010    | "Склад-СПб-Главный" | "доставлено" | 450    |
```

* **Типы данных:**

```python
move_id: str
warehouse_name: str
move_status: str  # "доставлено", "в пути", "задержка", "отменено"
capacity: int
```

2. **Считать исходный файл `warehouses.xlsx` в объект в DataFrame `warehouses_df`**

3. **Объединить `moscow_moves_df` и `warehouses_df` по `move_id`:**

```python
combined_df = JoinDataFrames(
    table1=moscow_moves_df,
    table2=warehouses_df,
    keys1=["move_id"],
    keys2=["move_id"],
    join_type="inner"
)
```

4. **Выбрать перемещения со статусом "задержка":**

```python
delayed_moves_df = FilterDataFrame(
    table=combined_df,
    filter_condition='move_status == "задержка"'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python

SaveExcel(file_path="delayed_moves.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по перемещениям с задержкой сохранена в delayed_moves.xlsx!")
```
