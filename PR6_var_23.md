# Вариант 23 - ExamResults: анализ результатов экзаменов и пересдач

---

## **Часть 1**

Компания **ExamResults** занимается анализом результатов экзаменов студентов в образовательных учреждениях.   
В системе регистрируются данные о сдаче экзаменов, полученных баллах и необходимости пересдачи.  
Руководство хочет контролировать студентов с низкими результатами, отслеживать экзамены, требующие пересдачи, и выявлять учащихся, нуждающихся в дополнительной поддержке.

Ваша задача как RPA-разработчика — автоматизировать работу с данными экзаменов через **DataFrame** в **Puzzle RPA**, отфильтровать записи по баллам, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

---

students_df["pass_percentage"] = students_df["exam_score"] / students_df["max_score"] * 100

## **Часть 2**

После фильтрации студентов с низкими баллами руководство хочет проанализировать необходимость пересдачи экзаменов.   
Существует второй файл `exams.xlsx`, в котором указаны данные о правилах пересдачи и статусах экзаменов.   
Нужно объединить данные и выделить экзамены, требующие обязательной пересдачи, а результат сохранить и показать пользователю.

---

1. **Создать файл `exams.xlsx` с 8 записями:**

```python
# exams.xlsx
| exam_name    | subject      | min_pass_score | retake_required | retake_date  | exam_type   |
|--------------|--------------|----------------|-----------------|--------------|-------------|
| "Математика" | "математика" | 40             | True            | 2026-03-20   | "основной"  |
| "Информатика"| "информатика"| 40             | True            | 2026-03-22   | "основной"  |
| "Физика"     | "физика"     | 40             | True            | 2026-03-25   | "основной"  |
| "Математика" | "математика" | 40             | False           | 2026-03-20   | "дополнительный"|
| "Информатика"| "информатика"| 40             | False           | 2026-03-22   | "дополнительный"|
| "Физика"     | "физика"     | 40             | False           | 2026-03-25   | "дополнительный"|
| "Химия"      | "химия"      | 40             | True            | 2026-03-28   | "основной"  |
| "Биология"   | "биология"   | 40             | True            | 2026-03-30   | "основной"  |
```

* **Типы данных:**

```python
exam_name: str
subject: str
min_pass_score: int
retake_required: bool   # требуется ли пересдача
retake_date: date
exam_type: str          # "основной", "дополнительный"
```

2. **Считать исходный файл `exams.xlsx` в объект в DataFrame `exams_df`**

3. **Объединить `low_score_df` и `exams_df` по `exam_name`:**

```python
combined_df = JoinDataFrames(
    table1=low_score_df,
    table2=exams_df,
    keys1=["exam_name"],
    keys2=["exam_name"],
    join_type="inner"
)
```

4. **Добавить расчет `score_gap`:**

```python
combined_df["score_gap"] = combined_df["min_pass_score"] - combined_df["exam_score"]
```

5. **Выбрать экзамены, требующие пересдачи:**

```python
retake_required_df = FilterDataFrame(
    table=combined_df,
    filter_condition='retake_required == True'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="retake_required.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по экзаменам, требующим пересдачи, сохранена в retake_required.xlsx!")
```
