## **Часть 1**

Компания **PayrollCheck** занимается управлением заработной платой и начислениями для сотрудников по городам России.   
Ежедневно обрабатываются данные о сотрудниках: часть получает высокую зарплату, часть — стандартную, а часть — требует пересмотра оклада.  
Руководство хочет контролировать сотрудников с зарплатой выше 80000 рублей, отслеживать начисления бонусов и выявлять сотрудников без премиальных выплат.

Ваша задача как RPA-разработчика — автоматизировать работу с данными о сотрудниках через **DataFrame** в **Puzzle RPA**, отфильтровать сотрудников по зарплате, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

2. **Считать исходный файл `employees.xlsx` в объект в DataFrame `employees_df`**

employees_df["is_high_salary"] = employees_df["salary"] > 80000

## **Часть 2**

После фильтрации сотрудников с высокой зарплатой руководство хочет проанализировать начисления бонусов.   
Существует второй файл `payroll.xlsx`, в котором указаны данные о бонусах и премиях.   
Нужно объединить данные и выделить сотрудников без бонусных выплат, а результат сохранить и показать пользователю.

---

1. **Создать файл `payroll.xlsx` с 8 записями:**

```python
# payroll.xlsx
| emp_id | bonus | premium | payment_date |
|--------|-------|---------|--------------|
| E001   | 15000 | 5000    | 2026-03-10   |
| E002   | 8000  | 3000    | 2026-03-10   |
| E003   | 0     | 0       | 2026-03-10   |
| E004   | 10000 | 4000    | 2026-03-10   |
| E005   | 0     | 0       | 2026-03-10   |
| E007   | 12000 | 5000    | 2026-03-10   |
| E008   | 25000 | 10000   | 2026-03-10   |
| E010   | 0     | 0       | 2026-03-10   |
```

* **Типы данных:**

```python
emp_id: str
bonus: int        # бонусная выплата
premium: int      # премия
payment_date: date
```

2. **Считать исходный файл `payroll.xlsx` в объект в DataFrame `payroll_df`**

3. **Объединить `high_salary_df` и `payroll_df` по `emp_id`:**

```python
combined_df = JoinDataFrames(
    table1=high_salary_df,
    table2=payroll_df,
    keys1=["emp_id"],
    keys2=["emp_id"],
    join_type="inner"
)
```

4. **Выбрать сотрудников без бонуса (bonus == 0):**

```python
no_bonus_df = FilterDataFrame(
    table=combined_df,
    filter_condition='bonus == 0'
)
```

5. **Сохранить результат в Excel и уведомить пользователя:**

```python
# Сохраняем Excel файл
SaveExcel(file_path="no_bonus_employees.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по сотрудникам без бонусов сохранена в no_bonus_employees.xlsx!")
```
