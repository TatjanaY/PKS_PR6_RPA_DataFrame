## **Часть 1**

Компания **FinanceTracker** занимается учётом финансовых операций и транзакций в крупных организациях.   
В системе регистрируются данные о доходах, расходах, переводах и других финансовых движениях.  
Руководство хочет контролировать крупные транзакции, отслеживать расходы по подразделениям и выявлять операции, требующие дополнительного анализа.

Ваша задача как RPA-разработчика — автоматизировать работу с финансовыми данными через **DataFrame** в **Puzzle RPA**, отфильтровать транзакции по сумме, добавить аналитические столбцы, сохранить результаты и уведомить пользователя.

```python
transactions_df["is_large"] = transactions_df["amount"] > 50000
```

```python
large_transactions_df = FilterDataFrame(
```

## **Часть 2**

После фильтрации крупных транзакций руководство хочет проанализировать расходы по подразделениям.   
Существует второй файл `departments.xlsx`, в котором указаны данные о подразделениях и их бюджетах.   
Нужно объединить данные и выделить транзакции типа "расход", а результат сохранить и показать пользователю.

---

* **Типы данных:**

```python

```

2. **Считать исходный файл `departments.xlsx` в объект в DataFrame `departments_df`**

3. **Объединить `large_transactions_df` и `departments_df` по `department`:**

```python
combined_df = JoinDataFrames(
    table1=large_transactions_df,
    table2=departments_df,
    keys1=["department"],
    keys2=["department"],
    join_type="inner"
)
```

4. **Добавить расчет `budget_usage_percent`:**

```python
combined_df["budget_usage_percent"] = combined_df["amount"] / combined_df["budget_limit"] * 100
```

5. **Выбрать транзакции типа "расход":**

```python
expense_transactions_df = FilterDataFrame(
    table=combined_df,
    filter_condition='transaction_type == "расход"'
)
```

6. **Сохранить результат в Excel и уведомить пользователя:**

```python

SaveExcel(file_path="expense_transactions.xlsx")

# Уведомляем пользователя
NotifyUser(message="Аналитика по крупным расходам сохранена в expense_transactions.xlsx!")
```
