## **Часть 1**

## **Часть 2**

Существует второй файл `payments.xlsx`, в котором указаны фактические платежи клиентов.

```

* **Типы данных:**

```python
order_id: str
payment_date: date
payment_amount: int
```

2. **Считать исходный файл `payments.xlsx` в объект в DataFrame `payments_df`**

3. **Объединить `moscow_orders_df` и `payments_df` по `order_id`:**

```python
combined_df = JoinDataFrames(
    table1=moscow_orders_df,
    table2=payments_df,
    keys1=["order_id"],
    keys2=["order_id"],

)
```

4. **Добавить расчет `remaining_amount`:**

```python
combined_df["remaining_amount"] = combined_df["amount"] - combined_df["payment_amount"]
```

5. **Выбрать просроченные заказы с задолженностью > 0:**

```python
late_unpaid_df = FilterDataFrame(

    filter_condition='remaining_amount > 0 and is_late == True'

```
